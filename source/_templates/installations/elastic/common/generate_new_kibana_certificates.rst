This guide assumes that a copy of ``certs.tar`` is placed in the root home folder (~/):

.. code-block:: console

  # mkdir /etc/kibana/certs
  # mv ~/certs.tar /etc/kibana/certs/
  # cd /etc/kibana/certs/
  # tar -xf certs.tar kibana.pem kibana.key root-ca.pem

.. End of include file
