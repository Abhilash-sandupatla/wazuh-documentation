  .. group-tab:: SysV Init

    Choose one option according to the OS used:

    a) RPM based OS:

      .. code-block:: console

        # chkconfig --add elasticsearch
        # service elasticsearch start
    
    b) Debian based OS:

      .. code-block:: console

        # update-rc.d elasticsearch defaults 95 10
        # service elasticsearch start
