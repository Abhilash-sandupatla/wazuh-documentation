Packages list
=============

This download page contains packages required for the Wazuh installation with the Wazuh version |WAZUH_LATEST|  :

- `Wazuh manager, Wazuh agent and Wazuh API`_

   - `AIX`_
   - `Linux`_
   - `HP-UX`_
   - `macOS`_
   - `Solaris`_
   - `Windows`_
- `Wazuh Kibana plugin and Wazuh Splunk plugin`_
- `Virtual machine`_
- `Elastic Stack`_

   - `Elasticsearch`_
   - `Kibana`_
   - `Filebeat`_
